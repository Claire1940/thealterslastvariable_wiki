<Section icon={Target} title="生死を分ける14日間の期限">

『オルターズ：最後の変数（The Alters: Last Variable）』において、コールドスリープ・チャンバーは単なる基地モジュールの一つではなく、科学者ヤンが生存するための絶対条件です。本作では、13年間に及ぶ過酷な「温暖期」が到来し、乗組員は強制的に人工冬眠（コールドスリープ）を強いられます。最初の重要目標であるクエスト「**強制休眠**」では、この施設を建設するために正確に14日間の猶予が与えられます。この期限に間に合わないと、チャンバーなしでは基地が地表の致命的な高熱から乗組員を保護できないため、即座にゲームオーバーとなります。

| 要求タイプ | 数値/詳細 | 重要度 |
| :--- | :--- | :--- |
| **クエスト名** | 強制休眠 | メインストーリーの節目 |
| **制限時間** | 14日間（サイクル1） | 厳格な期限 |
| **主な目標** | コールドスリープ・チャンバーの建設 | 生存に不可欠 |
| **副次目標** | 有機物備蓄の確保 | 乗組員の安全確保 |
| **失敗状態** | 放射線/熱による死亡 | 致命的なリスク |

この最初の2週間における時間管理は極めて重要です。資源採掘基地の拡張と、コールドスリープ・チャンバー建設のためのロジスティクスをバランスよく両立させなければなりません。任意で建設できる快適モジュールや初期の基地装飾に金属を使いすぎると、期限が迫ったときにチャンバーに必要な材料が不足する事態に陥る可能性があります。

<Callout type="warning" title="期限に関する警告">
14日間のタイマーを甘く見てはいけません。惑星の冷却期間は突如として終了します。チャンバーが完全に稼働し、電力が供給されていない場合、それまでの進行状況はすべて失われます。
</Callout>

<Section icon={Shield} title="建設コストと土壌のロジスティクス">

コールドスリープ・チャンバーを建設するには、地下基地のレイアウトを慎重に計画する必要があります。地上の構造物とは異なり、地下モジュールは土壌タイプによる制限を受けます。メイン電源と通路ネットワークに接続された、有効な配置セルを確保しなければなりません。チャンバーは占有面積が非常に大きいため、ゲーム序盤に不適切な配置をして拡張ルートを塞いでしまうと、後々高額な建て替え費用が発生することになります。

| 素材 | 必要量 | 主な入手先 |
| :--- | :--- | :--- |
| **金属** | 50 | 金属採掘基地（基地の南） |
| **ラピジウム** | 65 | ラピジウム鉱床（オアシス/南） |
| **電力** | 常時必要 | 基地パワーグリッド |
| **土壌タイプ** | 安定/標準 | 地下採掘 |

<CardGrid cols={2}>
  <Card title="戦略的な配置">
    チャンバーは**将来的な通路の拡張**が可能な場所に配置してください。後に研究所、ワークショップ、宿舎などのスペースが必要になるため、基地を「行き止まり」にしないようにしましょう。
  </Card>
  <Card title="資源のバッファ確保">
    チャンバー建設後も、常に**金属20のバッファ**を維持してください。地震によって施設が損傷した場合、機能を維持するために即座に修復キットが必要になります。
  </Card>
</CardGrid>

建設自体に労働時間が消費されるため、日中の時間帯にヤンまたは専門のオルター（物理学者など）をこの作業に割り当てるのが最善です。そうすれば、万が一資源が不足した場合でも、夕方に地表を探索して最後の数ユニットの金属やラピジウムを回収する余裕が生まれます。

<Callout type="tip" title="物理学者のボーナス">
建設とメンテナンスに**物理学者ヤン**を割り当てると、**効率が50%向上**し、より迅速にチャンバーを完成させることができ、期限を逃すリスクを減らせます。
</Callout>

</Section>

<Section icon={Activity} title="有機物管理と乗組員の規模拡大">

新規プレイヤーが陥りやすい最も一般的な失敗は、チャンバーの建設ではなく、チャンバーを稼働させる「燃料（有機物）」の確保に失敗することです。チャンバーに入る各人員は、13年間の睡眠中に生命を維持するために特定の量の**有機物**を必要とします。専門分野を持つオルターを増やすほど、必要な有機物の量は比例して増加します。

| 乗組員の構成 | 必要な有機物 | 推奨される生存バッファ |
| :--- | :--- | :--- |
| **科学者ヤンのみ** | 50 | 75 |
| **ヤン + オルター1人** | 100 | 130 |
| **ヤン + オルター2人** | 150 | 190 |
| **ヤン + オルター3人** | 200 | 250 |
| **ヤン + オルター4人** | 250 | 320 |

<Steps>
  <Step num="1" title="有機物採掘基地の設置">
    基地の背後に即座に有機物採掘基地を設置してください。これは金属採掘基地に次ぐ最優先事項です。
  </Step>
  <Step num="2" title="乗組員必要量の計算">
    新しいオルターを作成する前に、現在の有機物の備蓄量を確認してください。コールドスリープに必要なコストを支払えない場合は、新しい人格を実体化させてはいけません。
  </Step>
  <Step num="3" title="予備資源のロック">
    必要な有機物は「手を付けてはならない資金」として扱ってください。生存基準値を下回る場合は、温室の実験や贅沢な食料品に有機物を消費しないでください。
  </Step>
</Steps>

最終日に有機物が不足していることに気づいた場合、困難な選択を迫られます。地上の構造物を解体して資源を回収し、土壇場でかき集めるか、あるいは生存に必要な燃料が足りないことを知った乗組員による**オルターの反乱**のリスクに直面するかです。

<Callout type="success" title="生物学者の効率性">
**生物学者ヤン**は温室の作業効率を**100%向上**させるため、5人のフルメンバーに必要な大量の有機物を備蓄する上で不可欠な存在となります。
</Callout>

</Section>

<Section icon={Zap} title="野外研究所と外部への割り当て">

『オルターズ：最後の変数』におけるユニークなシステムとして、**野外研究所**があります。これらの施設は、岩の谷（Rocky Valley）や霧の峡谷（Misty Gorge）などのテラフォーミングされた地域に建設されます。オルターを野外研究所に割り当てると、彼らは温暖期の期間中ずっと地表に留まることになります。これはコールドスリープ計画において非常に重要な戦略的意味を持ちます。

**野外研究所の戦略的メリット:**
- **有機物コストゼロ**: 野外研究所にいるオルターはコールドスリープ・チャンバーを使用しないため、50の有機物コストを消費しません。
- **継続的な研究**: 科学者ヤンが眠っている間も、割り当てられた専門家は長期的なオアシスプロジェクトの研究を継続します。
- **資源の節約**: 専門家を外部に留まらせることで、少ない有機物備蓄でより多くの乗組員を維持できます。

<CardGrid cols={3}>
  <Card title="地質学者ヤン">
    **岩の谷（Rocky Valley）**の野外研究所に最適です。フェーズドラピジウムの研究を加速させ、次のサイクルのために地形の障害物を取り除きます。
  </Card>
  <Card title="化学者ヤン">
    **霧の峡谷（Misty Gorge）**に最適です。野外研究所の研究時間を10時間から約6時間15分に短縮します。
  </Card>
  <Card title="生物学者ヤン">
    ゲーム終盤の**滝（Waterfall）**の研究に不可欠です。最終段階のテラフォーミング層をアンロックするための生物サンプルを分析します。
  </Card>
</CardGrid>

ただし注意してください。野外研究所に駐在するオルターは**目に見えて老化**し、次の寒冷期が始まるまで基地のメンテナンスや人間関係のイベントに参加できなくなります。彼らを送り出す前に、個人のストーリーラインをできる限り進めておいてください。

<Callout type="info" title="上級者向けの戦略">
野外研究所を利用して、コールドスリープ・チャンバーで養う余裕のない専門家を「退避」させましょう。これにより、毎サイクル250の有機物を必要とすることなく、4つの専門家タイプすべてをアクティブに保つことができます。
</Callout>

</Section>

<Section icon={Map} title="最終日のチェックリストと目覚めのプロトコル">

寒冷期の最後の夜は、『オルターズ：最後の変数』において最も緊張する時間です。「コールドスリープに入る」ボタンをクリックすると、未完了のタスクはすべて失われるか、13年間遅れることになります。厳格なプロトコルに従うことで、目覚めたときに危機的状況ではなく、優位な立場からスタートできるようになります。

| アクション項目 | 優先度 | 理由 |
| :--- | :--- | :--- |
| **フェイザーの解体** | 高 | 惑星の熱による破壊を防ぐため |
| **負傷の治療** | 高 | 睡眠中の死亡リスクを下げるため |
| **野外研究所への割り当て** | 中 | 研究の進捗を最大化するため |
| **金属の備蓄** | 中 | 目覚めあとの修理に必要なため |
| **最後の会話** | 低 | モラルの低下や反乱を防ぐため |
